Bluecore
========

With this integration, Talkable syncs email opt-ins to Bluecore, facilitating efficient customer data management for targeted email marketing campaigns.

**Endpoint for Bluecore Integration:**
|br|
`POST <https://esp.talkable.com/bluecore>`_

**Params configuration for Bluecore integration**

.. code-block:: javascript

   const params = {
     email: "person-6ad487a3a0ccc1e7@talkable-sample.com", // required
     api_key: "6ad487a3a0ccc1e76ad487a3a0ccc1e7", // Bearer token
     campaign_id: "4783064988712960", // email campaign to trigger
     namespace: "missouri_quilt_co", // partner namespace
   };
