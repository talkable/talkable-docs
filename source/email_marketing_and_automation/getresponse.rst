GetResponse
===========

With this integration, Talkable syncs email opt-ins to GetResponse, streamlining customer data for email marketing campaigns.

**Endpoint for GetResponse Integration:**
|br|
`<https://esp.talkable.com/getresponse>`_

**Extras configuration for GetResponse integration**

.. code-block:: javascript

   const extras = {
     api_key: "xxxxxx", // endpoint URL TargetBay team should provide
     list_id: "32Jsds" // list id or campaign id
   };
